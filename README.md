## **Clothing Tracker : version 1.0**

Basic login building

JSON files for storage
 - easy read and write
 - minimal setup
 - initial learning

Data Model (Attributes captured)
 - Cloth ID
 - Cloth name (Visible in CLI)
 - Category (pant, shirt, shoes, socks) - pre-defined categories
 - wear count (Visible in CLI) = 0 (initial value)
 - last worn date (Visible in CLI) = today's date
 - last washed date (Visible in CLI) = date (enter manually)
