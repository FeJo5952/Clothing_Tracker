# **Clothing Tracker**
The clothing tracker is a personal tool that I created to manage my wardrobe and keep an organized set of clean clothes to wear everyday.


## **Clothing Tracker : version 1.0** - CLI Tool
Basic logic building

JSON files for storage

Data Model (Attributes captured) : class ClothingItem -->
 - Cloth ID
 - Cloth name (Visible in CLI)
 - Category (pant, shirt, shoes, socks) - pre-defined categories
 - wear count (Visible in CLI) = 0 (initial value)
 - last worn date (Visible in CLI) = None (intitial value), gets changed to the current date of each entry
 - last washed date (Visible in CLI) = date (enter manually)
